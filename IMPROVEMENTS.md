# Улучшения и новые функции (v1.3)

## 🐛 Исправленные ошибки

### 1. Дублирование кода в методе `_draw_device_details`
**Проблема**: Код для отображения пулов был дублирован несколько раз с неправильной логикой.
**Решение**: Перестроена логика отображения с правильной нумерацией строк и цветовым кодированием статуса.

### 2. Ошибка с переменной `sent` в методе `_update_device`
**Проблема**: Переменная `sent` использовалась до определения, что вызывало ошибку.
**Решение**: Переделана логика отправки прошивки:
- Файл читается полностью в память один раз
- Используется счётчик `sent` для отслеживания прогресса
- Добавлена правильная обработка таймаутов

### 3. Неполный метод `upload_via_http`
**Проблема**: Метод был незакончен и не использовался.
**Решение**: Удалён, так как прошивка отправляется через TCP.

### 4. Проблемы с диалогами ввода
**Проблема**: Диалоги не проверяли входные данные и могли вызвать исключения.
**Решение**: 
- Добавлена валидация всех входных данных
- Добавлена обработка исключений `ValueError` и `ValidationError`
- Добавлены информативные сообщения об ошибках в логи

### 5. Отсутствие обработки исключений
**Проблема**: Код не использовал специализированные классы исключений.
**Решение**: Добавлены 4 класса исключений:
```python
class DeviceError(Exception): pass
class ConnectionError(DeviceError): pass
class CommandError(DeviceError): pass
class ValidationError(DeviceError): pass
```

## ✨ Новые функции для Whatsminer M50S

### 1. Управление частотой (Клавиша `C`)

Установить частоту майнинга с автоматической валидацией (50-100%)

- Обновление статистики

### 2. Управление напряжением (Клавиша `V`)

## 🧪 Примеры использования

### Установить частоту на 85%
```python
monitor = DeviceMonitor(['192.168.1.100'])
monitor.selected_devices = {'192.168.1.100'}
await monitor.set_frequency(85)
```

### Получить информацию о чипах
```python
temps = await monitor.get_chip_temperatures('192.168.1.100')
print(temps)
```

### Создать резервную копию
```python
await monitor.backup_configuration('192.168.1.100')
```

### Изменить пароль
```python
monitor.selected_devices = {'192.168.1.100'}
await monitor.change_password('new_password_123')
```

## 📝 Логирование

Все операции логируются в `asic_monitor.log` с:
- Меткой времени
- Уровнем логирования (INFO, WARNING, ERROR, DEBUG)
- Описанием события
- IP адресом устройства (если применимо)
